Auto-boxing

- JS automatically wraps the string, numbers, boolean in an object like a box.

- This feature does not work with 'null' and 'undefined'

- Objects are references, when we assign an object to another, we also assign its address

- We can't compare objects directly, when we compare them, we actually compare the references

Destructuring object: an easier way to take properties out of an object
